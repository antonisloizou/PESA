# PESA Planner Web

Minimal HTML prototype based on `PESA_Complete_Planner_Final_Yellow.pdf`.

## Run locally

1. Install dependencies:
   `npm install`
2. Start the Node dev server (with hot reload):
   `npm run dev`
3. Open `http://127.0.0.1:8081`.

If you want a production build:

1. `npm run build`
2. `npm run preview`
